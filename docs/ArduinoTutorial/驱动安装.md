# 2.1 Windows系统USB驱动安装

 检查CP2102驱动是否已经安装

1. 用USB线连接计算机和ESP32。
   ![img-20230330142518](./media/img-20230330142518-1778304428184-4.png)

2. 进入计算机主界面，选择“**此电脑**”，右键单击选择“**管理**”。

   ![img-20230331145809](./media/img-20230331145809-1778304447392-6.png)

3.单击“**设备管理器**”。如果你的计算机已经安装了CP2102驱动，则可以看到“**Silicon Labs CP210x USB to UART Bridge(COMx)**”。

![img-20230331145950](./media/img-20230331145950-1778304464624-8.png)

 安装CP2102驱动

1. 如果未安装CP2102驱动，界面显示如下。

   ![img-20230331150247](./media/img-20230331150247-1778304499005-10.png)

2.单击“**CP2102 USB to UART Bridge Controller**”，右键选择“**更新驱动程序(P)**”。

![img-20230331150419](./media/img-20230331150419-1778304509434-12.png)

3.单击“**浏览我的电脑以查找驱动程序(R) **”.

![img-20230331150532](./media/img-20230331150532-1778304519233-14.png)

4.单击“浏览(R)...”选择CP210x_6.7.4(驱动路径：资料\USB驱动\CP2102 驱动文件-Windows\CP210x_6.7.4)，单击“**下一页**”

![image-20260509133236631](./media/image-20260509133236631.png)



5. 等待CP2102驱动安装完成。当界面显示如下时，表示已安装CP2102驱动。你可以关闭该界面。

   ![img-20230331151641](./media/img-20230331151641-1778304540457-18.png)

6.ESP32与计算机连接时，界面显示如下。

![img-20230331151718](./media/img-20230331151718-1778304551667-20.png)



# 2.2 MAC系统USB驱动安装

按照路径打开文件夹`资料\USB驱动\CP2102 驱动文件-MAC\macOS_VCP_Driver`，双击“SiLabsUSBDriverDisk.dmg”文件。

![img-20230329173728](./media/img-20230329173728.png)

可以看到以下文件。

![img-20230329173738](./media/img-20230329173738-1778306571109-27.png)

双击“Install CP210x VCP Driver”，勾选“Don’t warn me when opening application on this disk image”并单击“Open”。

![img-20230329173804](./media/img-20230329173804-1778306587755-29.png)

单击“Continue”。

![img-20230329173816](./media/img-20230329173816-1778306601251-31.png)

先点击“Agree”，然后点击“Continue”。



![img-20230329173824](./media/img-20230329173824-1778306615343-33.png)

继续点击“Continue”，然后输入你的用户密码

![img-20230329173832](./media/img-20230329173832-1778306661885-38.png)

![img-20230329173836](./media/img-20230329173836-1778306699636-40.png)

选择“Select Open Security Preferences”。

![img-20230329173845](./media/img-20230329173845-1778306736661-42.png)

点击安全锁，输入你的用户密码来授权。

![img-20230329173854](./media/img-20230329173854-1778306757625-44.png)

![img-20230329173858](./media/img-20230329173858-1778306773226-46.png)

看到锁被打开了，点击“Allow”。

![img-20230329173907](./media/img-20230329173907-1778306791832-48.png)

回到安装界面，根据提示等待安装.

![img-20230329173914](./media/img-20230329173914-1778306811409-50.png)

安装成功

![img-20230329173923](./media/img-20230329173923-1778306829332-52.png)
